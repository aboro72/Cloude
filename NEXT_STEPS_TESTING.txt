PLUGIN FEATURES

The Clock Preview plugin provides:

  - Real-time updating (every 1 second)

  - Numbers 1-12 on clock face

================================================================================
CRITICAL SUCCESS FACTOR
================================================================================

The hot-loading mechanism is the key innovation:
  ‚úÖ Activate plugin: No server restart needed
  ‚úÖ Deactivate plugin: No server restart needed
  ‚úÖ Update plugin: No server restart needed

This is achieved by:
  1. Modifying settings.INSTALLED_APPS at runtime
  2. Registering AppConfig in apps.app_configs
  3. Calling AppConfig.ready() to initialize hooks
  4. Reverse process for unloading with sys.modules cleanup

If tests confirm this works, the MVP is production-ready! üöÄ
